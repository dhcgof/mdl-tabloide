    🚀 Como Usar o Extrator Automático

    1

    Baixar o Script

    Baixe o arquivo extrair_produtos_individuais.py ou o arquivo EXTRAIR_PRODUTOS.bat (Windows)

    2

    Preparar o PDF

    Coloque o arquivo MDL TABLOIDE MENSAL (32).pdf na mesma pasta do script

    3

    Executar

    Windows: Duplo clique em EXTRAIR_PRODUTOS.bat
    Mac/Linux: Execute: python extrair_produtos_individuais.py

    4

    Resultado

    Uma pasta produtos_individuais/ será criada com 31 imagens: PRODUTO_01.jpg até PRODUTO_31.jpg

    ✅ Requisitos

    Para o Extrator Automático (Python):

    • ✓ Python 3.7+ instalado
    • ✓ Bibliotecas: pdf2image, Pillow
    • ✓ Poppler instalado (Windows)

    Para as Ferramentas Web:

    • ✓ Navegador moderno (Chrome, Firefox, Edge, Safari)
    • ✓ PDF.js já está incluído (CDN)
